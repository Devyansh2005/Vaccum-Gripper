# Vacuum Gripper Robotic Arm Simulation 🦾🌬️

![Vacuum Simulation Demo](https://vaccum-gripper.vercel.app/vite.svg)

A web-based 3D interactive simulation of a robotic arm equipped with a vacuum suction end-effector. Built with modern web technologies, this project provides a premium, responsive interface to explore robotic kinematics and pick-and-place operations involving a suction mechanism.

🌐 **Live Deployment**: [View Simulation Here](https://vaccum-gripper.vercel.app/)

## Features ✨

- **High-Fidelity 3D Environment**: Powered by Three.js with realistic lighting, shadows, and orbit controls.
- **Vacuum Suction Mechanism**: Features a conical suction cup end-effector capable of picking up and dropping paper objects dynamically.
- **Accurate Kinematics**: 6-DOF configuration allowing full range of motion.
- **Interactive Teleoperation**: A modern glassmorphism UI dashboard allows you to manipulate all axes in real-time and toggle suction.
- **Premium Aesthetics**: Designed with a sleek dark mode and vibrant accents for a modern engineering tool feel.

## Tech Stack 🛠️

- **3D Rendering**: [Three.js](https://threejs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Core**: Vanilla JavaScript, HTML5, and CSS3

## Running Locally 🚀

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Devyansh2005/Vaccum-Gripper.git
   cd Vaccum-Gripper
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
